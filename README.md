## Getting Started

Should add .env back into the project

First, run the development server:

```bash
pnpm i

pnpm run dev

```

To check the built version

```bash
pnpm build

pnpm start
```

## Tests

This repository comes with `jest`

Run unit tests

```bash
pnpm test
```
